# ascii-games
A collection of ASCII terminal mini-games written in Python with `curses`.
Work in progress 

## Status
🚧 Under construction — first game (snake-like) in progress

## Requirements
- Python 3.x
- Linux / macOS

## Running
```bash
git clone https://github.com/0xMarta/ascii-games
cd ascii-games
python main.py
```
